Data Sources

Below is a list of the countries, agencies, and sources who have either made their data publicly available or provided us their data for inclusion in the API. For details on how to interact with these datatsets, see the get started page or visit the reference documentation.

Canada
Years: 1948 - 2019 | Fires: 404,601

Canada Wildfire Information System National Fire Database (CWFIS-NFDB)

The CWFIS dataset includes fires from 1948 to 2019 across all provinces, territories (except the Nunavut territory), and Parks Canada. It is updated annually. Since we require fires in our dataset to have at least a start date, we eliminated 3,383 fire records (0.008%) that account for 6.2M acres burned (0.019%). We also intersected Parks Canada fires with province/territory boundaries provided by the Canada Census to get the actual province/territory the fire occurred in. This dataset is considered the authoritative source for fire occurrence in Canada.

Canadian Forestry Service. 2019. See https://cwfis.cfs.nrcan.gc.ca/ha/nfdb for more information.

Chile
Years: 1984 - 2019 | Fires: 202,605

Corporación Nacional Forestal Fire Database (CONAF-FDB)

The CONAF dataset includes fires from 1984 to 2019 for all regions in Chile. This dataset is very complete with the exception of latitude/longitude information for fires prior to 2002. These fires have GEOREF coordinate information and we are currently working to convert these but there is missing information on the major and minor grids needed to complete the conversion. We hope to fix this with the agency provider. This dataset is considered the authoritative source for fire occurrence in Chile.

Chile Corporación Nacional Forestal. 2019. See http://www.conaf.cl/incendios-forestales/incendios-forestales-en-chile/estadisticas-historicas/ for more information.

México
Years: 2010 - 2017 | Fires: 62,081

México Comisión Nacional Forestal Centro Nacional de Manejo del Fuego Fire Database (CONAFOR-FDB)

The CONAFOR dataset includes fires from 2010 to 2017 for all Méxican states. There are some fires with missing latitude/longitude data but each fire should have at least a state. Since we require fires in our dataset to have at least a start date, we eliminated 45 fire records (0.0007%) that account for 1,113 acres burned (0.0001%). Aside from these issues, the CONAFOR dataset is very complete with ample data on sizes, dates, and causes. This dataset is considered the authoritative source for fire occurrence in México.

México Comisión Nacional Forestal. 2018. See https://datos.gob.mx/busca/dataset/incendios-forestales for more information.

United States
Years: 1992 - 2015 | Fires: 1,880,545

Fire Program Analysis - Fire Occurrence Database (FPA-FOD)

This dataset lags behind by up to two years to account for data submission, cleaning, and processing from the fifty states. Data from 1992 to 2015 across all federal land agencies, states, and some tribal governments are available. This dataset is considered the authoritative source for fire occurrence in the United States.

Short, Karen C. 2017. Spatial wildfire occurrence data for the United States, 1992-2015 [FPA_FOD_20170508]. 4th Edition. Fort Collins, CO: Forest Service Research Data Archive. https://doi.org/10.2737/RDS-2013-0009.4
